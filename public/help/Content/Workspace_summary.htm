Summary

This topic should open from the context-sensitive link workspace_summary.

Selecting a particular workspace brings up the summary of the recent activity associated with that workspace. You can also view and edit the data, and the work files associated with it.

Customizing your Summary view

The Summary tab defaults to show All Activity. You can also read insights from the owner of the workspace about a dataset or work file.

Using your Summary tab

From this tab, you can perform the following tasks:

Edit Workspace: This presents the Workspace Settings screen. on this screen, you can edit the following options:

  • Workspace Name: You may want to edit the name of a workspace for a variety of reasons, such as providing clarity, or labeling your workspaces. You can also make your workspace private by unchecking the Publicly available option.
  • Summary: Add a short description about the workspace. You can format your summary using the text options.
  • Workspace Image:Select a PNG, GIF, or JPG image and click Addlocated below the Workspace Image field. The Add option switches to Change, once you have uploaded an image.
  • Sandbox location: Unless you create a sandbox location, this field displays the location as None.
  • Change owner: You can delegate ownership of one of your workspaces to another member of your team. If you are an admin, you may have set up workspaces for the members of the team and need to give them access.
  • Workspace status: You can archive a workspace, it is only available in read-only mode.

Delete this Workspace: Once you select Delete Workspace, this action cannot be undone.

Add or Edit Members: You can add more members to the workspace to help with the collaborative process. They can add files, import data, and receive messages about the workspace.

Add a Sandbox: Brings up the Add a Sandbox dialog. Using drop-down options, you can select an instance, database, and schema to associate with your sandbox. Each user can only be associated with a single sandbox.

Add a note: Add a note to comment on a particular workspace.You have the option to add a desktop file, work file, or dataset to add support your note. You can also remove any attachments and notify selected members on your project team about your activity.

Add an insight: Adding an insight is useful for all Chorus users,and appears on Home dashboard. You can add a desktop file, work file, or dataset to add support your insight. You can also remove any attachments and notify selected members on your project team about your activity.
