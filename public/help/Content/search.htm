Search

The Search page displays the results of your query.

See the Searchable Chorus fields table below for details on what information is indexed and therefore searchable.

Note that instances are indexed by default daily with the instance owner's credential at midnight. The time and frequency can be configured by a Chorus administrator, as described in the Chorus Installation Guide.

Chorus search also supports a typeahead search that matches on names of objects.

Features of the Search page

  • Viewing the results of your search
  • Using the options on the Search page

Viewing the results of your search

When you submit a search string from any of the pages, you are presented with the Search page and will notice the following:

  • Defaults to Search in All of Chorus and Show All Results.
  • Displays all the recent activity associated with the search object.

Note that results are rolled up into primary Chorus objects such as datasets, users, and so on.

Using the options on the Search page

Search in: Use this drop-down menu to narrow the scope of your search to your own files by selecting My Workspaces.

Show : Use this drop-down menu to narrow your search to look within a specific object group. Here are the options:

  • Work Files
  • HDFS Files
  • Desktop File Attachments directly to Notes/Insights
  • Datasets
  • Instances
  • Workspaces
  • People

Note that wildcards are supported.
