Data

This topic should open from the context-sensitive linkdatasets.

The Data tab displays the list of datasets associated with the workspace that you have chosen to explore. The Show option displays the total number of datasets.

Click on a particular dataset to view details. See "Displaying your data" for more information.

Customizing your Data tab view

The Data tab defaults to the Show option, all datasets. The other drop down options you can select are as follows:

  • source datasets
  • Chorus views
  • sandbox datasets

Adding data to your source table

The Data tab shows the activity stream for the selected dataset. The activity stream shows the name of the user and their activity with the dataset. You can perform the following tasks:

Add a note: Add a note to comment on a particular dataset.You have the option to add a desktop file, work file, or dataset to add support your note. You can also remove any attachments and notify selected members on your project team about your activity.

Import Now: You can import any well formatted .csv text files.

Create Import Schedule: You can create a schedule for importing data from a source to a destination table. You can customize the imported data, such as requesting truncation, limiting the number of rows, and setting a recurrent import schedule.

Preview Data:You can preview the data in a particular dataset.
