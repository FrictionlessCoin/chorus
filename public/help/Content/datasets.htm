Data

Data is the second tab in the workspace folder. The Data tab displays the list of datasets associated with the workspace that you have chosen to explore.

Features of the Data tab

  • Sorting and viewing datasets
  • Exploring a particular data source

Sorting and viewing datasets

  •  Browse the list of associated datasets in the left pane. The Data tab defaults to the Show all datasets: option. Selecting one of the other viewing options from the drop-down list determines how you can work with the files presented to you.These options are as follows:
    • source datasets: Presents source tables and views that can be added to a workspace by associating them from the instance.
    • Chorus Views: Provides a dataset that Chorus recognizes. It is usually a SQL statement that Chorus can visualize and import. You can edit your Chorus view. Select the Edit button to bring up the code mirror interface. Edit the file, and select run. You can cancel your changes, or save and return to column view mode.
    • sandbox datasets: Displays the tables and views present in the chosen database or schema.
  • Select a particular dataset to view details. See "Working with your data sources" for more information.

Exploring a particular data source

Select the data source that you wish to explore. This brings you to the Data detail page.

  • Explore: Select this option to search for columns within the dataset.
  • Derive: Use this option to create a Chorus view. You can preview your SQL statement, add a join and select columns.
  • Visualize: Select this option to create a visual representation based on your data. You can also add data filters. The types of charts you can use to visualize your data are as follows:
  • Frequency chart
  • Histogram
  • Heatmap
  • Time Series
  • Boxplot
  • Data Preview:You can preview the data in a particular dataset. The preview fetches one hundred rows of data.
