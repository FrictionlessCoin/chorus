Home

This topic should open from the context-sensitive link dashboard.

When you log in to Chorus, you will see the Home page. You will use this page to access the main areas of Chorus.

Features of the Home page

  • Displays recent activity.
  • Displays the total number of Chorus users.
  • Provides access to and displays the number of insights.
  • Lists and provides access to workspaces.
  • Lists and provides access to instances and datasets.

Using the Home page

You can use the Home page to perform the following major functions:

Manage your account: Select Chorus Users, located at the top of the right pane, to get to see the list of usernames, sorted last name first. You can reset the list to sort by first name.The list also shows which users are administrators. From this page,selecting Add User presents the New User dialog. See "Users"

  • Browse Recent Activity: This is located on the left pane and shows a complete list of any recent activity by a user, or of any data changes. Also allows you view all the insights that have been created.
  • Select Browse Data to view all information about instances and datasets: This is located on the right pane. This section of the Home page provides information about available instances. You can perform the following from the Home page:
  • Add an Instance: Presents the Create a New Instance dialog. You can register either an existing Greenplum Database instance or a Hadoop Distributed File System.
  • View all Instances:Presents a list of instances. See "Instances"
  • Select My Workspaces to view all information about workspaces. This is located on the bottom of the right pane and lists all the existing workspaces. From this area of the Home page, you can Create a new workspace or select Browse All to browse all existing workspaces. See "Workspaces".See "Work Files"
