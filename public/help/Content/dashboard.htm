Home

The Home page allows you to explore and edit the major information in Chorus, whether it is related to the users, instances and datasets, or workspaces. The following topic presents a list of features that describe the information you can access and tasks you can perform from this page.

Features of the Home page

  • Editing your profile
  • Using search
  • Viewing the list of Chorus users
  • Exploring recent activity
  • Exploring Instances
  • Exploring workspaces

Editing your profile

Use the drop down under your username in the top right pane to perform the following:

  • Your Profile: Select this option to edit it. See "Editing your Profile"
  • Sign Out: Use this option to exit Greenplum Chorus.

Using Search

Type the name of a dataset, workspace, or user in the Search field. Accessing the Search function from this page, automatically defaults to Search in All of Chorus and Show All Results. For more information, See "Search"

Viewing the list of Chorus users

Select Chorus Users for a list of users. See "Users"

Exploring Recent Activity

Browse the list presented in the left pane which displays a complete list of recent user activity, and data changes.

  • Insights: Presents the list of insights. You can browse through all the user comments against workspaces and the insights associated with them.
  • Comment: Allows you to comment on a workspace. You can also delete your comment. If you own the workspace, you can promote your comment to an insight. If you are an administrator, you can promote another user's comment to a insight and publish it for the benefit of all Chorus users.

Exploring Instances

  • Add an instance: You can create a new instance on a registered Greenplum or Hadoop instance. Presents the Create a New Instance dialog. See "Instances"
  • View All: Select this to bring up the Instances page, where you can view a list of instances. See "Instances"

Exploring Workspaces

View the list of workspaces. Each one shows the recent activity such as comments, insights, and timestamp

  • Create a new workspace: Allows you create a new workspace either as a public or private space.
  • View All:Presents the Workspaces page. See "Workspaces".
