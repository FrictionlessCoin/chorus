Home

The Home page allows you to explore the major information in Chorus, whether it is related to the users, instances and datasets, or workspaces. You can use the Homepage to view recent activity in Chorus and quickly navigate to datasets and workspaces of interest.

As you navigate away from your Home page, breadcrumbs appear across the top of the page to inform you of your location, and allow you to quickly navigate around your path.

The following list describes the information you can access and tasks you can perform from this page.

Features of the Home page

  • Editing your profile
  • Using search
  • Viewing the list of Chorus users
  • Exploring recent activity
  • Exploring instances
  • Exploring workspaces
  • Viewing your notifications

Editing your profile

Use the drop down under your username in the top right pane to perform the following:

  • Your Profile: Select this option to edit it. See "Editing your Profile"
  • Sign Out: Use this option to exit Greenplum Chorus.

Using Search

Type the name of a dataset, workspace, or user in the Search field. Accessing the Search function from this page, automatically defaults to Search in All of Chorus and Show All Results. For more information, See "Search"

Viewing the list of Chorus users

Select Chorus Users for a list of users. See "Users"

Exploring Recent Activity

Browse the list presented in the left pane which displays recent user activity and data changes. This list is filtered to display global activity (for example activity related to instances), and activity on workspaces to which, you, the current user, are a member. Types of activities are:

  • Insights: These are Insights that have been concluded about the data. Use the links at the top right to filter the Recent Activity pane to show only Insights, or All Activity. You and other users can add comments to an insight, and publish the insight to Chorus to make it visible to all Chorus users. If you see an additional label named Unpublish beneath an insight, that insight has already been published. Any member of the workspace can unpublish an insight.
  • Other activities: Other activities can be Notes, Edits, Imports, Associations, and so on. If you own the workspace, you can promote note to an insight, if you are an administrator, you can promote other users' note to insights.

 

Click Comment to add a comment to any activity, including insights.

Exploring Instances

In the right pane of the Home page you can view the instances to which you have permissions, either via a shared account, or via credentials you provided.

  • Add an instance: Click this to open the Create a New Instance dialog where you can add a new instance on a registered Greenplum Database or Hadoop instance.
  • View All: Select this to open the Instances page, where you can view a list of all instances. See "Instances"

Exploring Workspaces

In the right pane of the Home page, beneath the list of instances, you can view your Workspaces. The Workspaces you can see here are all the active (not archived) workspaces to which you are a member.

Each Workspace shows associated recent activity such as comments, insights, and timestamp.

  • Create a new workspace: Click this to open the Create a New Workspace dialog where you can create a new workspace either as a public or private space.
  • View All:Select this to open the Workspaces page, where you can view all public workspaces, and can also choose to view archived public workspaces. See "Workspaces".

Viewing Your Notifications

Notifications are presented to you through the notifications alert box. The notifications alert box is in the top right Chorus menu bar and is red in color if you have any unread notifications. Click the alert box to open a message dialog that displays the top five notifications, click Show all messages to navigate to the Notification page to see all the notifications sent to you.
