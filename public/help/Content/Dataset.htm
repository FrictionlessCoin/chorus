Displaying your data

This topic should open from the context-sensitive linkdataset.

The Dataset Detail tab displays details about a particular dataset. It displays the following information:

  • Lists the names of the columns: you can also use the Search field to find any particular column.
  • Displays the total number of columns.
  • Shows the type of data stored in each column.
  • Provides you a preview of the table.

Using the Dataset detail tab

You can customize the data display as follows:

  • Derive: Provides options to create a Chorus view. You can preview your SQL statement, add a join and select columns.
  • Visualize: This option allows you create any one of the following types of charts based on your data. You can also add data filters. The types of charts you can use to visualize your data are as follows:
  • Frequency chart
  • Histogram
  • Heatmap
  • Time Series
  • Boxplot

Adding data to your source table

The Data tab shows the activity stream for the selected dataset. The activity stream shows the name of the user and their activity with the dataset. You can perform the following tasks:

Add a note: Add a note to comment on a particular dataset.You have the option to add a desktop file, work file, or dataset to add support your note. You can also remove any attachments and notify selected members on your project team about your activity.

Import Now: You can import any well formatted .csv text files.

Create Import Schedule: You can create a schedule for importing data from a source to a destination table. You can customize the imported data, such as requesting truncation, limiting the number of rows, and setting a recurrent import schedule.

Preview Data:You can preview the data in a particular dataset.
