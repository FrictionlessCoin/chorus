Working with your data sources

The Dataset detail tab displays details about a particular dataset.

Features of the Datasets page

  • Working with a sandbox dataset
  • Working with source datasets and Chorus views

Working with a sandbox dataset

You can only use the sandbox dataset to explore. You can import data into it and use it as a destination, but cannot use it as a source. As such you can only perform the following functions:

  • Import File: import files from source tables and views.
  • Add a note
  • Preview Data

Working with source datasets and Chorus views

You can perform the following tasks when working in the Chorus view

  • Import File: import files from source tables and views.
  • Add a note
  • Import Now: to import data into a new or existing table. You can limit the number that are being imported.
  • Create import schedule: to import data into a new or existing table, limit the number of rows, and schedule a start and end time.
  • Preview Data
