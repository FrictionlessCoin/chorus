Workspaces

The Workspaces page displays the complete list of existing workspaces.

Features of the Workspaces page

  • Viewing the list of workspaces
  • Creating a workspace
  • Exploring a particular workspace

Viewing the list of workspaces

  • Explore the workspaces in the list, the default view is Active Workspaces. This shows all the currently active workspaces. It displays information about the workspace such as the owner, the summary, and whether it is private or public.
  • Change the default to sort by All Workspaces. This shows all currently active and archived workspaces. Archived workspaces have restricted / read-only functionality.
  • Select a workspace to view its summary. See "Summary"

Creating a workspace

Create workspace: Use this option to create and name your workspace. You can also choose to make your new workspace public or opt to keep it private. Only workspace members can view private workspaces. Non-workspace members can browse public workspaces in read-only mode.

Exploring a particular workspace

Select the workspace that you wish to explore. This brings you to the workspace that contains the following tabs:

  • Summary: Provides an overall view of the workspace. See "Summary"
  • Data: Provides information about all the data associated with the workspace. See "Data"
  • Work Files: Provides a list of all the files associated with the workspace. See "Work Files"
