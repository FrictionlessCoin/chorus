Users

This topic should open from the context-sensitive link users.

The Users page displays a list of users. This page provides information about Chorus users. If you are logging in for the first time, use the Add User link to create a new account. You can also use this link to create multiple accounts.

Features of the Users page

  • Lists users by last name, but you can modify the search to sort by first name.
  • Displays the total number of users.
  • Select Add Users to create a new account.
  • Display the users that have "Administrator" privileges.
  • Provides you with the ability to add a new user.
  • Starting point for accessing your page and account.

Permissions Model

Currently Chorus supports a simple permissions model and can only provision users and administrator accounts.

A user is a member of Chorus and can own objects.

You can only implement Active Directory or application wide settings.
