Work Files

This topic should open from the context-sensitive link workfiles.

The Work Files tab displays the work file list page. On the right pane,the tab shows the activity stream for the currently highlighted file. The activity stream shows the name of the user, and the name of the file added to the workspace. Your work file draft is auto-saved every 30 seconds.

Customizing the Work Files view

You can customize your view options as follows:

  • Show: The default is All files. The other drop down options you can select are as follows:
  • SQL files
  • Code files
  • Text files
  • Other files
  • Sort: The default is Alphabetically. You can also sort By Date.

Using the Work Files tab

  • Upload file: Browse and upload a file. Note that the configurable limit is 250 MB. When you set # Maximum upload work file size, you can use 'M', or 'G' as a short cut to specify megabyte and gigabyte in the chorus.properties file. For example: chorus.workfile.max_file_size=250M.
  • Create SQL File: Allows you to create a blank SQL file. You can also upload your own SQL file.
  • Copy latest version to another workspace:Brings up the Copy File to Workspace dialog. You can either search for a workspace or select one from a list.
  • Delete: Deletes the file from your workspace.
  • Download: Download any files that you have uploaded or any version of the SQL files.
  • Add a note:Add a note to comment on a particular work file.You have the option to add a desktop file, work file, or dataset to add support your note. You can also remove any attachments and notify selected members on your project team about your activity.
