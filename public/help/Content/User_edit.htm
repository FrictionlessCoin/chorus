Editing your Profile

The Profile page displays account information such email address, lists workspaces, and shows recent activity.

Features of the Profile page

Managing your own profile

Managing user accounts

Managing your own profile

You can perform the following tasks to manage your own accounts:

Edit Profile: This allows you to modify all the information you entered when you created the user account.

Change password: This allows you to change your password. This link is only visible if you are exploring your own account.

Managing user profiles

You can perform the following tasks if you are an Administrator:

Edit Profile: This allows you to modify all the information you entered when you created the user account.

Delete User: This allows you to delete an account. You must transfer all active workspaces and instances before you can delete it.
