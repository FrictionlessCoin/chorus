Workspaces

The Workspaces page displays the complete list of existing workspaces.

Features of the Workspaces page

  • Viewing the list of workspaces
  • Creating a workspace
  • Exploring a particular workspace

Viewing the list of workspaces

  • Explore the workspaces in the list, the default view is Active Workspaces. This shows all the currently active workspaces. It displays information about the workspace such as the owner, and the summary.
  • Change the default to sort by All Workspaces. This shows all currently active and archived workspaces. Since this display mode also shows archived workspaces, your functionality is restricted.
  • Select a workspace to view its summary. See "Summary"

Creating a workspace

Create workspace to create and name your workspace. You can also choose to make your new workspace public or opt to keep it private. If you unselect the check box and choose to keep your workspace private, it will appear in the Active Workspaces list with a Private label.

Exploring a particular workspace

Select the workspace that you wish to explore. This brings you to the workspace folder which contains the following tabs:

  • Summary: Provides an overall view of the workspace. See "Summary"
  • Data: Provides information about all the data associated with the workspace. See "Data"See "Working with your data sources"
  • Work Files: Provides a list of all the source files associated with the workspace. See "Work Files"
