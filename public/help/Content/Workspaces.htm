Workspaces

This topic should open from the context-sensitive link workspaces.

The Workspaces page displays the complete list of exiting workspaces that associated with you. It also displays the total number of workspaces . From this page, you can select Show to customize your view. You can also create a new workspace. The se tasks allow you to perform the following options:

Features of the Workspaces page

  • Lists the workspaces based on the filter selected from the Show option.
  • Displays the total number of workspaces based on the filter selected from the Show option.
  • Displays the owner of each workspace.
  • Select Create Workspace to create a new account.
  • Select a workspace to edit or view information.

Using the Workspaces page

You can perform the following tasks from the Workspaces page:

Show:Using this function, sort the associated workspaces to customize your view.

  • Active Workspaces: This is the default and shows all the currently active workspaces associated with you.
  • All Workspaces: Shows all currently active and archived workspaces associated with you. Since this display mode also shows archived workspaces, your functionality is restricted.

Create workspace: Select this option to create and name your workspace. You can also choose to make your new workspace public or opt to keep it private. If you unselect the check box and choose to keep your workspace private, it will appear in the Active Workspaces list labeled Private.
